# SwitchVsIf
SwitchVsIf 在Android 手机上的 性能测试


我们有时会遇到这样的程序员，经常在选择  Switch 或 if else  语句的问题上纠结，有人认为if else 的执行效率更高，本程序模似了一段测试代码，
大家可以能过本程序直观的感受两者的差别。  建议使用低频手机测试。 
事实上，两者一定各有优缺点，大家可以在使用中各有取舍。
